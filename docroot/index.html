No Password

Introduction

A new style of authentication is becoming popular. Instead of passwords an email is sent to the person trying to log in with a link which contains a token. Following the link logs the person in. This system depends on the person maintaining control of their email address, but means they no longer have to invent strong passwords and websites no longer have to store passwords.

This website is a test bed for a proposed app which will automate the process.

Instructions

Registration

In order to log in to the site, first you need a user name and associated email address. Follow the "No Password Registration" link below. On that page you can enter that data. Hitting the "Register" button will send an email to the address. There is a link in that email. Following the link will log you into the site.

Log in

The "No Password Login Request" link below will let you log into the site. On that page you can enter your user name. Hitting the "Request login" button will send an email to the address. There is a link in that email. Following the link will log you into the site.

Application

Data

Your user name and email address is displayed, along with the number of times you have logged into the site.

Contact

This button will allow you to control whether or not you will be contacted in future about this project, including crowd funding campaigns.

Delete

This button will mark your data deleted.

Log out

This button will log you out.
